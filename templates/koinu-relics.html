Koinu Relics

Verifiable historical artifacts for Dogecoin transactions.

How Ownership Works

  • Each inscription is a distinct artifact on-chain, even if two inscriptions reference the same transaction and Merkle proof.
  • So yes, anyone can technically mint the same source transaction again unless a protocol-level claim rule is enforced.
  • If you want Dogemap-style uniqueness, define and index a canonical claim rule: "first valid inscription for (txid, blockhash) wins".

Theme Logic

  • Regular transactions render in Dogecoin gold / shiba palette.
  • Coinbase-like transactions render in dense pink solo-miner style.
  • Both are still verified from /r/tx and /r/blockinfo.
